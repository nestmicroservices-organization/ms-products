# Product Microservice

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Dev

1. Clone Repository

2. Install dependencies 

```bash
$ npm install
```

3. Create a File `.env` with environment variables based in `env.template`

4. Execute prisma migration `npx prisma migrate dev`

5. Run Aplication

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
